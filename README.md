# 🏠 Elsesser & Co. - Real Estate Agency Website

> Полнофункциональный веб-сайт агентства недвижимости для Екатеринбурга

## 📋 О проекте

**Elsesser & Co.** — современная платформа для продажи и аренды недвижимости в Екатеринбурге с полным циклом взаимодействия клиентов, агентов и администраторов.

### Ключевые возможности:
- 🏘️ Каталог готового жилья (продажа и аренда)
- 🏗️ База новостроек от застройщиков с планировками
- 👥 Многоуровневая система ролей (user/agent/admin)
- 💬 Встроенный чат для общения с агентами
- ⭐ Система избранного и сравнения объектов
- 📊 Аналитика и CRM для агентов
- 📱 Адаптивный дизайн для всех устройств

## ✨ Основной функционал

### 🏠 Для посетителей сайта (незарегистрированных):

#### Каталог недвижимости:
- **Готовое жильё на продажу** — квартиры в Екатеринбурге с готовыми документами

- **Умный поиск** по названию, адресу, району
- **Фильтры**:

  - Район Екатеринбурга
  - Близость к метро

- Полное описание на русском языке
- Галерея фотографий (с возможностью увеличения)
- Основные характеристики:
  - Площадь (общая, жилая, кухня)
  - Этаж / этажность дома
  - Количество и тип санузлов
  - Балкон/лоджия
  - Тип ремонта
  - Год постройки дома
- Инфраструктура и удобства
- Расположение на карте
- Информация о районе

#### Дополнительно:
- Страница «О компании» с информацией об агентстве
- Страница «Контакты» с формой обратной связи
- Подписка на новостную рассылку

## 📊 Структура базы данных

### Основные таблицы:

#### Пользователи и роли:
- **`users`** — все пользователи системы
  - Поля: id, email, password (hash), role (user/agent/admin), created_at
  - Связи: автор объектов, отправитель сообщений, владелец избранного

#### Недвижимость:
- **`properties`** — готовое жильё (продажа и аренда)
  - Основные поля: title_ru, description_ru, price, area_sqft (м²), bedrooms
  - Адрес: location, community, district
  - Характеристики: rooms_type, bathroom_type, balcony, renovation, floor/total_floors
  - Дом: house_type, build_year, has_elevator, has_garbage_chute
  - Транспорт: metro_station, minutes_to_metro
  - Для аренды: deposit, prepayment, utilities_included, rent_terms
  - Связи: agent_id → users, images, amenities

- **`property_images`** — фотографии объектов
  - Поля: property_id, image_path, is_main (главное фото)
  - Связь: property_id → properties

- **`new_buildings`** — жилые комплексы (новостройки)
  - Основные: name_ru, developer_id, district, price_from, price_per_sqm
  - Сроки: completion_quarter, completion_year, construction_stage
  - Характеристики: building_class, floors, sections, parking_type, finishing_type
  - Медиа: video_url, tour_3d_url
  - Связи: developer_id → developers, images, layouts

- **`new_building_layouts`** — планировки в ЖК
  - Поля: building_id, room_type (студия/1-комн/2-комн...), price_from, area_from
  - Связь: building_id → new_buildings

#### Справочники:
- **`amenities`** — удобства (кондиционер, мебель, WiFi, бытовая техника...)
- **`property_amenities`** — связь объектов и удобств (M:N)
- **`ekb_districts`** — районы Екатеринбурга (Центр, Уктус, Академический...)
- **`developers`** — застройщики (имя, контакты, рейтинг)
- **`infrastructure_items`** — инфраструктура рядом (школы, магазины, парки...)

#### Взаимодействие:
- **`favorites`** — избранные объекты пользователей
  - Связь: user_id → users, property_id → properties
  
- **`messages`** — чат между клиентами и агентами
  - Поля: sender_id, receiver_id, message, is_read, created_at
  - Связи: sender_id → users, receiver_id → users

- **`inquiries`** — заявки клиентов
  - Поля: name, email, phone, property_id, message, status (pending/processing/completed)
  - Связь: property_id → properties

- **`viewings`** — календарь просмотров
  - Поля: user_id, agent_id, property_id, scheduled_at, status
  - Связи: user_id → users, agent_id → users, property_id → properties

- **`reviews`** — отзывы на объекты и агентов
  - Поля: user_id, property_id, agent_id, rating (1-5), comment, is_approved
  - Связи: user_id → users, property_id → properties, agent_id → users

#### Дополнительно:
- **`newsletter_subscribers`** — подписчики на email-рассылку

## 🚀 Быстрый старт

### Требования
- **PHP**: 8.0+ (рекомендуется 8.1)
- **MySQL**: 8.0+ или MariaDB 10.3+
- **Web Server**: Apache 2.4+ с mod_rewrite
- **OSPanel** или аналогичный локальный сервер (XAMPP, Laragon)

### Установка

1. **Клонировать репозиторий**
```bash
git clone https://github.com/your-username/elsesserandco-site.git
cd elsesserandco-site
```

2. **Создать базу данных**
```bash
# Через MySQL CLI
mysql -u root -p < database/create_database.sql

# Или импортировать через phpMyAdmin
# database/create_database.sql
```

3. **Настроить подключение к БД**

Создать файл `includes/config/database.php` (по образцу):
```php
<?php
define('DB_HOST', 'localhost');
define('DB_NAME', 'realestate_db');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_CHARSET', 'utf8mb4');

function getDBConnection() {
    try {
        $dsn = "mysql:host=" . DB_HOST . ";dbname=" . DB_NAME . ";charset=" . DB_CHARSET;
        $pdo = new PDO($dsn, DB_USER, DB_PASS);
        $pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
        return $pdo;
    } catch(PDOException $e) {
        error_log("Database connection error: " . $e->getMessage());
        die("Connection failed. Please check database configuration.");
    }
}
?>
```

4. **Запустить сервер**
```bash
# OSPanel: запустить через GUI
# Или встроенный PHP сервер (для тестирования):
php -S localhost:8000
```

5. **Открыть в браузере**
```
http://localhost:8000
# или
http://elsesserandco-site.local
```

## 🔮 Планы развития

### В разработке:
- 🗺️ Интеграция с Яндекс.Картами для отображения объектов
- 📧 Email-уведомления для агентов и клиентов (SMTP)
- 📊 Расширенная аналитика для агентов (конверсия, популярные объекты)
- 🔔 Push-уведомления о новых сообщениях
- 🌐 REST API для мобильного приложения
- 💳 Онлайн-оплата резервирования просмотров
- 🤖 Чат-бот для первичной консультации

### Рассматривается:
- Интеграция с Авито/ЦИАН для импорта объектов
- Виртуальные 3D-туры по квартирам
- Ипотечный калькулятор
- Блог и новости рынка недвижимости
